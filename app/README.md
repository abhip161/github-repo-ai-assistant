# GitHub Repo AI Assistant 🚀

This project builds an AI-powered assistant that can understand and answer questions about any GitHub repository.

## 🔥 Features

* Download and process GitHub repositories
* Extract markdown documentation
* Prepare structured data for AI models

## 🛠️ Tech Stack

* Python
* Requests
* Zipfile
* Frontmatter

## 📦 Project Structure

```
github-repo-ai-assistant/
│
├── app/
│   └── ingest.py
├── README.md
└── requirements.txt
```

## ▶️ How to Run

```bash
python app/ingest.py
```
